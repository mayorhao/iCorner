一款jQuery插件
可以使页面卷起一角

简介

iCorner能够在页面右下角制造卷角效果，就像本页一样

通过使用CSS的border属性来制造三角形，通过三角形的叠加来实现视觉效果，同时也可以给卷角加上链接功能，无需使用图片，卷角的 大小可调

它是一个 超轻量级的JavaScript插件，只有仅仅1.2KB噢！ 马上下载！

使用方法

第一步，需要在项目中引入jQuery.js与icorner.js文件

<script src="http://code.jquery.com/jquery-1.10.2.min.js" type="text/javascript"></script>
<script src="iCorner.js" type="text/javascript"></script>

假设我们需要在id为 paper 的div加上卷角效果，我们只需要这么做：

HTML:

<div id="paper"></div>

JavaScript:

$('#paper').iCorner();

参数设置

插件有四个插件可选，但您也可以直接使用默认设置:

size: 40;
id: undefined;
link: undefined;
background: '#fff';

比如:

$('#paper').iCorner({
    size: 25,
    background: 'black'
});

$('#paper').iCorner({
    background: '#ffcc00',
    link: 'about.html',
    id: 'about'
});

size

size可选，能够调整卷角的大小，默认为40px

id

可选，选择是否为corner加上id属性.如果id为undefined，那么corner将不会有Id属性

link

可选，使corner被点击时，打开一个新窗口指向设置的链接,link应设置成字符串，像这样："http://idouyu.com".

background

强烈建议您根据页面不同的背景颜色，设置不同的background值，这样才能够使卷角更加自然，在这个页面中，因为页面背景为灰色，所以，我将background 设置为"#ccc"，这样卷角效果毫无违和感
